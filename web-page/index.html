Bootcamp 2021: Running Man

My project consists of stickmen. The main stickman is what the player will play as. The aim of the game is to simply jump over the stickmen known as "Killers". If the player hits the "killer" then the game will end and will display the message "Game Over." There is no end to this game, it will go on forever until the "killer" hits the player. The requirement to play the game is just the spacebar.
